# PracticeAccountingApp

Приложение для учёта и управления данными студентов, групп и практик. Реализовано на WPF с использованием архитектурного паттерна MVVM и CommunityToolkit.Mvvm.

## Описание

Проект представляет собой десктопное приложение для работы с учебными данными. Основной акцент сделан на разделение логики и представления, удобную работу с коллекциями данных и расширяемость архитектуры.

Приложение поддерживает работу со студентами, группами и связанными сущностями, а также предоставляет интерфейс для поиска, редактирования и отображения данных в реальном времени.

## Технологии

- .NET (WPF)
- C#
- MVVM (CommunityToolkit.Mvvm)
- ObservableCollection
- DispatcherTimer (debounce / UI-логика поиска)
- XAML

## Архитектура

Проект использует паттерн MVVM:

- **Views** — XAML страницы и окна
- **ViewModels** — логика представления, команды, биндинги
- **Models** — модели данных
- **Services** (если используются) — работа с данными / бизнес-логика

Для связи UI и логики используется data binding и генерация через CommunityToolkit.Mvvm:

- `[ObservableProperty]`
- `[RelayCommand]`

## Основной функционал

- Просмотр списка студентов и связанных сущностей
- Поиск с задержкой (debounce через DispatcherTimer)
- Добавление и редактирование записей через диалоговые окна
- Обновление интерфейса через ObservableCollection
- Навигация между страницами приложения

## Особенности реализации

Поиск реализован через таймер, чтобы избежать частых обновлений UI при вводе текста.

ObservableCollection используется для автоматического обновления интерфейса при изменении данных без ручного refresh.

CommunityToolkit.Mvvm снижает количество boilerplate-кода в ViewModel за счёт source generators.

## Требования

- Windows 10/11
- .NET Desktop Runtime (под версию проекта)

## Запуск

1. Клонировать репозиторий
2. Открыть `.sln` в Visual Studio
3. Установить зависимости NuGet (если не подтянулись автоматически)
4. Запустить проект (`Start`)
