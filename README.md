## PracticeAccountingApp

Приложение для учёта и управления данными студентов, групп и практик. Реализовано на WPF с использованием архитектурного паттерна MVVM и CommunityToolkit.Mvvm.

Описание

Проект представляет собой десктопное приложение для работы с учебными данными. Основной акцент сделан на разделение логики и представления, удобную работу с коллекциями данных и расширяемость архитектуры.

Приложение поддерживает работу со студентами, группами и связанными сущностями, а также предоставляет интерфейс для поиска, редактирования и отображения данных в реальном времени.

Технологии
.NET (WPF)
C#
MVVM (CommunityToolkit.Mvvm)
ObservableCollection
DispatcherTimer (для UI-логики, например поиска)
XAML (UI-разметка)
Архитектура

Проект построен по MVVM:

Views — XAML-окна и страницы
ViewModels — логика представления, команды и биндинги
Models — сущности данных
Services (при наличии) — работа с данными и бизнес-логика

Для взаимодействия UI и логики используется биндинг свойств и команд через ObservableProperty и RelayCommand.

Основной функционал
Отображение списков студентов и связанных данных
Поиск по данным с задержкой (debounce через DispatcherTimer)
Редактирование и добавление сущностей через диалоговые окна
Обновление UI через ObservableCollection без ручного ререндера
Навигация между страницами приложения
Особенности реализации

Поиск реализован через таймер, чтобы избежать лишних обновлений UI при вводе текста. Данные привязаны через ObservableCollection, что обеспечивает автоматическое обновление интерфейса при изменениях коллекции.

Команды и свойства генерируются через CommunityToolkit.Mvvm, что уменьшает количество шаблонного кода в ViewModel.
